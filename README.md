# Giorgi Tsertsvadze — Portfolio

Personal portfolio website built with pure HTML, CSS, and JavaScript. 
Cyberpunk/dark-tech aesthetic with advanced canvas animations.

## Features
- Animated background — floating particles, wave lines, hexagons, scanline sweep
- Matrix-style canvas with mouse interaction
- Glitch effect hero section with typed text animation
- Fully working contact form via EmailJS — sends directly to Gmail
- Auto-reply email sent to the visitor upon contact
- Custom animated cursor with trailing ring
- Scroll-reveal animations on all sections
- Responsive design for mobile and desktop
- Deployed on Netlify via GitHub

## Sections
- Hero — name, typed roles, stats counter, profile photo
- About — personal info + live terminal animation
- Experience — freelance, Skytel, EPAM
- Skills — animated progress bars across 4 categories
- Projects — 6 projects with tech tags
- Education — Vilnius University, MicroOne, freeCodeCamp, EPAM
- Contact — EmailJS form + Gmail + LinkedIn + GitHub links

## Tech Stack
`HTML` `CSS` `JavaScript` `EmailJS` `Netlify` `GitHub`
